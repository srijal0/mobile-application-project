# Trendora

Trendora is a full-stack fashion e-commerce mobile application developed using Flutter for the frontend and a RESTful Web API for the backend. The application allows users to register, log in, browse fashion products, view product details, add items to a shopping cart, and place orders.

The project follows the MVVM (Model-View-ViewModel) architectural pattern to maintain clean separation between UI, business logic, and data layers. It also incorporates local data persistence using Hive and SharedPreferences, along with automated unit and widget testing to ensure reliability, maintainability, and code quality. The app is fully responsive and supports both phone and tablet screen sizes.
